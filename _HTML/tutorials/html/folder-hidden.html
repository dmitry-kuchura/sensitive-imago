Расположение ./markup/hidden

Директрия в которой находяться файлы с которых будут скомпилированны файлы, для примера аяксовых запросов.

Как правило тут лежат файлы с контентами для всплывающих окон - формы и прочее


Все *.ejs файлы с данной директории компилируються в ./dist/hidden со сменной расшрирения на *.php
смотри gulp задачу Gulp docs / Пространства имен / gulp / gulp ejs:hidden


Все остальные файлы, к примеру *.json, будут просто переброшенны без измененний в ту же директорию ./dist/hidden
смотри gulp задачу Gulp docs / Пространства имен / gulp / gulp ejs:hidden:static
